# Dasar-Dasar Pemrograman (DDP)

Repository ini berisi kumpulan **tugas, latihan, dan proyek** dari mata kuliah **Dasar-Dasar Pemrograman (DDP)**.  
Tujuannya adalah untuk mendokumentasikan proses belajar konsep dasar pemrograman dan penerapannya menggunakan bahasa pemrograman tingkat awal.

## Deskripsi Mata Kuliah
Mata kuliah **Dasar-Dasar Pemrograman** membahas konsep fundamental dalam pemrograman komputer, mulai dari logika dasar, struktur data sederhana, hingga implementasi kode.  
Mahasiswa diharapkan memahami cara berpikir komputasional dan mampu menulis program yang efisien serta terstruktur.

## Tujuan Pembelajaran
1. Memahami konsep algoritma dan logika pemrograman.  
2. Mengenal tipe data, variabel, operator, dan struktur kontrol.  
3. Mampu membuat program dengan perulangan dan percabangan.  
4. Menggunakan fungsi, array, dan struktur data dasar.  
5. Menyusun kode yang rapi, efisien, dan mudah dibaca.
